Vous allez créer une unique page WEB dont la vocation est d'être la landing page de l'IUT Sénart Fontainebleau

Une landing page est la première page d'un site, la page d'atterrissage qui peut être commune ou distincte de la page d'accueil ...

Vous devrez utiliser des propriétés de transformation géométriques en CSS ainsi que des transitions ou animations CSS pour créer des effets en mouvement ou au survol.

En particulier vous devrez utiliser la propriété CSS transform et utiliser des courbes de Bézier pour les transitions et animations.

Vous pouvez aussi utiliser du Javascript ou p5.js si vous le souhaitez.

Voici un exemple :

http://but2.mmi-iutsf.org/R315/

Cette page doit être la plus attractive et esthétique possible pour donner envie au visiteur d'aller plus loin ...

RENDU :

Une archive au format .zip ou .rar à votre nom qui contiendra tous les fichiers nécessaires à votre page.
